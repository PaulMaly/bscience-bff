# XRPD data management GUI: Backend for Frontend

### Requirements

- `node -v >= 15`
- `npm -v >= 7`
- Postgres (any relatively new)

## Installation

```bash
cp env.ini.sample env.ini
npm install
node db_seed.js
```

## Running

```bash
npm run
```
